Ghidra NSpawn

Ghidra-NSpawn
The Ghidra SRE tool in a self contained environment.

Isolated Software Reverse Engineering Environment…

Ghidra is a Software Reverse Engineering (SRE) environment developed by the U.S. National Security Agency and released as Open Source.

This project builds a custom Ghidra environment that lives within a systemd-nspawn container environment, isolating Ghidra and the software under inspection from the surrounding system.

  • Based on systemd-nspawn containers.
  • Combine with VMs for safely conducting SRE research on malware.
